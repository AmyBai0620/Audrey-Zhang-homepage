# Audrey Zhang 个人主页系统 - 功能完善计划

## 📋 需求分析总结

### 用户画像
- **目标用户**：混合人群（学生、学术同行、公众）
- **使用场景**：联系咨询、了解研究、查看课程、申请加入研究组
- **更新频率**：每周更新（高频）
- **网站风格**：现代简约风格（清爽、易读）

### 核心需求
1. **高频更新** - 需要便捷的内容管理
2. **多样化访客** - 需要清晰的信息架构
3. **学术展示** - 需要专业的内容呈现
4. **互动交流** - 需要联系和留言功能
5. **国际化** - 需要中英文支持
6. **移动优先** - 需要独立的手机版设计

CREATE TABLE project_update (

    project_id BIGINT NOT NULL,

    update_date DATE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (project_id) REFERENCES research_project(id)
);

-- 招募信息表
CREATE TABLE recruitment (
    id BIGSERIAL PRIMARY KEY,
    professor_id BIGINT NOT NULL,
    project_id BIGINT,
    position_type VARCHAR(50) NOT NULL, -- MASTER, PHD, POSTDOC
    title VARCHAR(200) NOT NULL,
    description TEXT,
    requirements TEXT,
    deadline DATE,
    status VARCHAR(20) DEFAULT 'OPEN', -- OPEN, CLOSED
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (professor_id) REFERENCES professor(id),
    FOREIGN KEY (project_id) REFERENCES research_project(id)
);
